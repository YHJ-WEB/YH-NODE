第一步：使用font-face声明字体 
@font-face {font-family: 'projectIconfont';
    src: url('projectIconfont.eot'); /* IE9*/
    src: url('projectIconfont.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('projectIconfont.woff') format('woff'), /* chrome、firefox */
    url('projectIconfont.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('projectIconfont.svg#projectIconfont') format('svg'); /* iOS 4.1- */
}
第二步：定义使用projectIconfont的样式 
.projectIconfont{
    font-family:"projectIconfont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
